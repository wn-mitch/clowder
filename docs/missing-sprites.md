# Missing Sprites — Living Placeholder Tracker

Everything in this list is currently rendering as a colored square, a tinted
`white_pixel`, or a reused sprite from another system. Update this file every
time a new placeholder is introduced or an existing one is replaced.

## Convention

Whenever you add a placeholder (anything rendered via `white_pixel` tint, a
solid color, or a shared-with-something-else sprite), **add an entry below**
in the appropriate section. When real art is wired in, strike through the
entry and move it to [Resolved](#resolved) with the commit/PR hash.

Entry format:

```
- **<Thing>** — <what it looks like today>. `<file>:<line>`
```
